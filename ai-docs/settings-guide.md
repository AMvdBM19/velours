# Settings Guide

## Settings Sections

### Agency Identity
- Agency name, tagline, contact email, phone
- Some fields may be locked after initial wizard setup

### Financial Settings
- Base rate per 30 min — Default pricing for bookings
- Worker payout % — Percentage of booking total paid to worker
- Agency share % — Remainder after worker payout
- Currency (EUR/USD/GBP)
- Tax rate % and tax label (e.g., BTW 21%)

### Booking Rules
- Default slot duration (minutes)
- Minimum lead time (hours before booking)
- Maximum advance booking days
- Offline behaviour (hide_worker / show_unavailable)
- No-show revenue policy (full / partial / none)

### Client Settings
- Client approval mode: auto (immediate) or manual (agent review required)

### Branding
- Primary and secondary colors (used in widget and UI)
- Logo URL
- Widget layout (grid / list / minimal)

### Integrations
- WhatsApp Business API — API key, phone number ID, business account ID. Leave empty to disable WA notifications.
- Maps API key — Enables address autocomplete in booking form
- LLM API key — Powers the AI assistant

## Locked Settings
Some settings are locked after the initial setup wizard is completed. These show a "Locked" badge and cannot be changed by the agent. Contact support to modify locked settings.

## How to Change Settings
1. Go to Settings page
2. Select the section from the left navigation
3. Edit fields (unlocked ones only)
4. Click "Save Changes" button (shows count of modified fields)
