# /ai-docs/ — AI Assistant Knowledge Base

Markdown documentation files loaded by the AI assistant to answer agent questions.
Updated after each phase as features are built.

## Files

- `erp-reference.md` — ERP navigation and modules overview
- `booking-guide.md` — booking flow, manual bookings, statuses
- `worker-guide.md` — profiles, availability, offline toggle
- `client-guide.md` — approval flow, blacklist, ratings
- `finance-guide.md` — revenue calculations, BTW, exports
- `settings-guide.md` — all settings and what each does
- `tools-reference.md` — what the AI assistant can and cannot do
- `changelog.md` — feature changelog, appended per phase
